# ConsoleApp

Простое консольное приложение на Python для демонстрации работы с Git.

## Описание

Приложение выводит приветствие и показывает пример функции `greeting`.

## Инструкция по запуску

1. Убедитесь, что установлен Python 3.
2. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Legend8883/python-for-college.git
   cd ConsoleApp
3. Запустите приложение:
    ```bash
   python main.py
